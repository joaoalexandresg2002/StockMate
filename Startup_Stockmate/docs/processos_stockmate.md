# Processos de Qualidade e Checklist de Code Review

## Checklist mínimo para revisão de PRs:
    A. Código legível e com comentários quando necessário.
    B. Testes relevantes adicionados ou atualizados.
    C. Nenhuma quebra nas funcionalidades existentes.
    D. Mensagem de commit clara.
    E. Seguiu a DoD (docs/dod.md).
