# Product Backlog — StockMate

    Prioridade: Alta

    Critérios de aceitação:

        A. Permite nome, quantidade e preço

        B. Valida campos obrigatórios

        C. Salva corretamente no banco ou arquivo

## 2. Como operador, quero remover produtos para retirar itens obsoletos.

    Prioridade: Alta

    Critérios:

        A. Aviso antes de excluir

        B. Não permitir exclusão de produto inexistente

## 3. Como operador, quero atualizar informações de produtos.

    Prioridade: Média

    Critérios:

        A. Atualiza nome, quantidade e preço

        B. Valida entradas

## 4. Como operador, quero visualizar o estoque atual.

    Prioridade: Alta

    Critérios:

        A. Lista produtos ordenados

        B. Exibe quantidade total

## 5. Como gestor, quero ver relatórios simples do estoque.

    Prioridade: Baixa
