# StockMate

Sistema simples de controle de estoque em Python — Projeto de Engenharia de Software III.

## Como executar (modo rápido)

    1. Crie um virtualenv (recomendado)

        bash
        python -m venv .venv
        source .venv/bin/activate   # Linux / macOS
        .venv\Scripts\activate      # Windows (PowerShell)

    2. Instale dependências (se houver)

        pip install -r requirements.txt

    3. Execute o sistema

        python src/main.py
