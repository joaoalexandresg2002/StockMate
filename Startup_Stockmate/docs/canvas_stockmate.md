# Canvas — StockMate

## 1. Problema

    A. Pequenos comerciantes e iniciantes não possuem ferramentas simples para controle de estoque.

    B. Anotações manuais geram erros, perdas de produtos e falta de organização.

    C. Sistemas profissionais são caros, complexos ou exigem conhecimento técnico avançado.

## 2. Segmentos de Clientes

    A. Pequenos comerciantes e microempreendedores.

    B. Estudantes que querem aprender lógica de programação aplicada.

    C. Usuários que desejam uma ferramenta simples para organizar produtos.

    D. Pessoas sem conhecimento em desenvolvimento web ou sistemas avançados.

## 3. Proposta de Valor Única
    “Um sistema simples, rápido e acessível para controlar estoques usando apenas Python e linha de comando.”

    O StockMate oferece praticidade, baixo custo e facilidade ao realizar operações essenciais de forma clara e eficiente.

## 4. Solução

    A. Cadastro de produtos.

    B. Atualização de informações (quantidade, nome, preço).

    C. Remoção de produtos.

    D. Listagem completa do estoque.

    E. Busca por ID ou nome.

    F. Interface simples via menu interativo no terminal.

## 5. Canais

    A. Executável diretamente pelo terminal do computador.

    B. Arquivos hospedados no GitHub.

    C. Documentação incluída no próprio repositório para facilitar uso e estudo.

## 6. Fluxo de Receita

    Projeto acadêmico — Não monetizado.

    Custos inexistentes, já que o sistema utiliza ferramentas gratuitas.

## 7. Estrutura de Custos

    A. Tempo de desenvolvimento.

    B. Ambiente Python (gratuito).

    C. Editor de código (VSCode).

    D. Documentação do projeto (GitHub Pages).

## 8. Métricas

    A. Número de produtos cadastrados.

    B. Redução de erros no controle de estoque.

    C. Facilidade de uso percebida pelos usuários.

    D. Execução sem travamentos e sem erros.

    E. Correção das funcionalidades de acordo com os requisitos.

## 9. Vantagem Injusta

    A. Simplicidade extrema: qualquer pessoa consegue usar.

    B. Código enxuto e fácil de estudar.

    C. Ideal como ferramenta educacional para iniciantes em Python.

    D. Totalmente offline e gratuito.
